# GUI для Universal Stream Video Downloader

`gui.py` добавляет графическую оболочку поверх текущего загрузчика на `yt-dlp`. Через интерфейс можно выполнить все основные операции без ручного редактирования командной строки:

- загрузить существующий локальный `settings.toml`;
- изменить все рабочие параметры;
- сохранить конфиг в текущий файл или в новый;
- выбрать папку выгрузки, файл cookies, файл со ссылками и архив скачанного;
- запустить скачивание и смотреть лог прямо в окне.

Минимальная поддерживаемая версия интерпретатора: Python 3.10+.

В репозитории также есть шаблоны `urls.example.txt` и `cookies.example.txt`, если нужно быстро подготовить связанные файлы для `urls_file` и `cookie_file`.

## Запуск

После установки зависимостей запустите:

```powershell
python gui.py
```

На Linux команда та же:

```bash
python gui.py
```

## Что есть в интерфейсе

### Верхняя панель

- выбор файла конфигурации;
- загрузка конфига;
- сохранение изменений;
- сохранение в новый TOML;
- сброс формы к значениям по умолчанию.

### Вкладка `Общие`

- папка выгрузки;
- файл архива скачанных видео;
- флаг `continue_on_error`.

### Вкладка `Скачивание`

- формат `yt-dlp`;
- формат склейки;
- шаблон имени файла;
- параметры повторных попыток и таймаутов;
- флаги `noplaylist`, `quiet`, `write_thumbnail`, `write_info_json`.

### Вкладка `HTTP`

- `Cookie`;
- `cookie_file`;
- `User-Agent`;
- `Referer`;
- дополнительные заголовки в формате `Header: Value`.

### Вкладка `Ссылки`

- ссылки по одной на строку;
- серые примеры ввода в пустом поле, чтобы было легче понять формат;
- кнопка `Вставить из буфера` для вставки ссылок без `Ctrl+V`;
- путь к отдельному файлу со ссылками;
- строки, начинающиеся с `#`, можно использовать как комментарии.

### Нижняя часть окна

- кнопка запуска скачивания;
- очистка лога;
- статус выполнения;
- живой лог скачивания.

## Рекомендуемый сценарий работы

1. Откройте `gui.py`.
2. Если нужно, сначала создайте `settings.toml` из `settings.example.toml`, либо просто откройте GUI и сохраните новый конфиг из формы.
3. При необходимости создайте локальные `urls.txt` и `cookies.txt` из `urls.example.txt` и `cookies.example.txt`.
4. Заполните параметры на вкладках.
5. При необходимости сохраните конфиг.
6. Нажмите `Запустить скачивание`.
7. Следите за прогрессом и итогом в логе.

## Особенности

- GUI запускает скачивание в фоне, поэтому окно не зависает во время работы.
- Конфиг по умолчанию ищется рядом со скриптом или рядом с `exe`, если приложение собрано через `PyInstaller`.
- Если `settings.toml` отсутствует, GUI сначала попробует загрузить `settings.example.toml`, а затем откроется с базовыми значениями по умолчанию.
- Относительные пути интерпретируются относительно папки выбранного конфига.
